# 3d-print-skill — Gemini CLI

This plugin provides the `3d-print` skill for parametric 3D model design and printing.

## Skill Activation

When the user asks to design or print a 3D part, activate the `3d-print` skill using the `activate_skill` tool.

## Tool Mapping

| Claude Code Tool | Gemini CLI Equivalent |
|------------------|----------------------|
| Read | read_file |
| Write | write_file |
| Edit | edit_file |
| Bash | shell |
| Glob | glob |
| Grep | grep |
